 ## kindle与ipad的比较
 ### kindle优势
 1. 墨水屏
 2. 续航时间长
 3. 比较小，容易携带，随时阅读
 4. kindle适合的场景时小说和漫画
 
 ### ipad优势
 1. 生态丰富
 2. 屏幕大
 3. 更适合技术类书，对排版要求比较高
